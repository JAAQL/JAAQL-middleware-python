Welcome to the JAAQL Console!
Please use the following commands
- /switch {db_name} - Switches your current database
- /list - Lists the databases you have access to
- /file - Open a file prompt so you can upload a script file
- /help - Shows a list of commands you have access to
- /logout - Logs you out
- /clear - Clears the console
- /clearhis - Clears the console history
- /refresh - Refreshes your database access
